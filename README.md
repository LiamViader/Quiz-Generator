Welcome to **Quiz AI Generator**, a full-stack application that leverages Artificial Intelligence to create customized quizzes based on any topic you provide. Whether you're a student building study tools or just looking for some trivia fun, this app has you covered!

## Key Features

- **AI-Powered Generation**: Instantly generate quizzes using OpenAI's GPT models.
- **Google OAuth Integration**: Secure and easy login with your Google account.
- **Hybrid Rate Limiting**:
  - **Anonymous Users**: Share a global pool of 10 uses per day.
  - **Logged-in Users**: Get a personal quota of 5 guaranteed quizzes daily, plus access to the global pool once the personal quota is exhausted.
- **Public Quiz Gallery**: Explore and take quizzes created by the community.
- **Responsive Design**: Modern and clean UI built with Material UI, fully responsive for all screen sizes.

## Tech Stack

### Frontend
- **Framework**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Material UI (MUI)](https://mui.com/)
- **State Management**: React Context API
- **Authentication**: [@react-oauth/google](https://www.npmjs.com/package/@react-oauth/google)
- **HTTP Client**: [Axios](https://axios-http.com/)

### Backend
- **Framework**: [NestJS](https://nestjs.com/) (TypeScript)
- **Database**: [MongoDB](https://www.mongodb.com/) (using [Mongoose](https://mongoosejs.com/))
- **AI Integration**: [OpenAI API](https://openai.com/api/)
- **Authentication**: [Passport.js](https://www.passportjs.org/) with JWT and Google Strategy

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v24.12.0 or higher recommended)
- [MongoDB](https://www.mongodb.com/try/download/community) (local or Atlas)
- OpenAI API Key
- Google Cloud Console Project (for OAuth credentials)

### Step 1: Clone the Repository

```bash
git clone https://github.com/LiamViader/Quiz-Generator.git
cd Quiz-Generator
```

### Step 2: Configure Environment Variables

#### Backend (`/backend/.env`)
Create a `.env` file in the `backend/` directory with the following:
```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
OPENAI_API_KEY=your_openai_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_secret
```

#### Frontend (`/frontend/.env`)
Create a `.env` file in the `frontend/` directory:
```env
VITE_GOOGLE_CLIENT_ID=your_google_client_id
VITE_API_URL=http://localhost:3000
```

### Step 3: Installation & Running

#### Setup Backend
```bash
cd backend
npm install
npm run start:dev
```

#### Setup Frontend
```bash
cd frontend
npm install
npm run dev
```

## Usage

1. Open the app in your browser (usually `http://localhost:5173`).
2. (Optional) Log in with Google to get your personal daily quota.
3. Enter a topic in the generator field.
4. Customize the number of questions and difficulty.
5. Hit **GENERATE** and enjoy your custom quiz!
6. You can also visit the **Explore** section to see quizzes generated by others.
